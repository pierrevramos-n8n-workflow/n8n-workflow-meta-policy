Política de Privacidade

Aplicativo: n8n workflow
Última atualização: 09 de fevereiro de 2026

1. Sobre o Aplicativo

O aplicativo n8n workflow é utilizado para gerenciamento de páginas, ativos comerciais e automações integradas à plataforma Meta (Facebook e Instagram), por meio das APIs oficiais.

2. Dados Coletados

O aplicativo pode acessar, mediante autorização do usuário:

  • Nome e ID do perfil
  • Endereço de e-mail (quando permitido)
  • Lista de páginas administradas
  • Informações públicas das páginas
  • Métricas básicas de desempenho
  • Tokens de acesso fornecidos pela Meta para autenticação

O acesso ocorre exclusivamente dentro das permissões concedidas pelo próprio usuário.

3. Finalidade do Uso

Os dados são utilizados exclusivamente para:

  • Gerenciamento e administração de páginas
  • Execução de automações autorizadas
  • Integração com sistemas operacionais internos

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros para fins comerciais.

4. Armazenamento e Segurança

Adotamos medidas técnicas adequadas para proteger os dados contra acesso não autorizado. Os tokens de acesso são utilizados apenas para execução das funcionalidades do aplicativo.

5. Exclusão de Dados

O usuário pode solicitar a exclusão de seus dados a qualquer momento enviando e-mail para:
pierrervramos@gmail.com

Os dados serão removidos em até 30 dias após a solicitação.
